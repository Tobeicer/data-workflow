# 闲鱼来源适配器

状态：目录契约已建立，采集实现尚未开始。

## 用途

补充二手流通、存量设备、价格和公开需求线索，不作为高可信主数据的直接覆盖来源。

## 公开/授权边界

只处理公开商品信息或人工明确授权的数据。不得采集私聊、账户、订单、地址、会员、支付或后台信息，不得绕过登录、签名和风控。

## 停止条件

遇到授权缺失或到期、登录要求、验证码、滑块、403/429、权限或签名限制、来源条款不明确、页面结构变化或异常请求量时立即停止并保留证据，转人工评估。

## 晋级标准

完成人工授权记录、合规小样本、脱敏 fixtures、解析与契约测试、统一 CLI 和 `--dry-run`、L0-L2、变化检测、质量门禁与 `run_result.json` 后才能从 `planned` 晋级；连续稳定运行验收前不得启用 n8n。
