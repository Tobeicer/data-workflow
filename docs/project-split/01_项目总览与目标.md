# 游艺圈项目总览与目标

版本：V0.1  
日期：2026-07-02

---

## 1. 项目定位

游艺圈是面向游戏游艺行业的 AI 数据引擎平台，服务对象包括个人采购用户、电玩城经营者、厂家、厂家业务员、周边服务商、达人、业务精英、5G 电玩运营商和行业内容生产者。

项目核心不是做传统黄页，也不是单纯资讯站，而是将行业中分散、混乱、过期、真假难辨的信息，整理成可搜索、可验证、可推荐、可询价、可交易、可沉淀的数据资产。

一句话定位：

> 游艺圈，游戏游艺行业的 AI 数据引擎。

---

## 2. 核心价值

| 价值 | 说明 |
|---|---|
| 看得见，查得到 | 厂家、产品、文审、政策、资料、商机、配件和服务都能查 |
| 拍得准，找得到 | 通过图片、游戏画面、二维码、铭牌识别设备和供应商 |
| 配得准，买得省 | AI 根据预算、地区、场地和政策生成选品配置单 |
| 卖得快，撮合准 | 商机、AI 代询价、AI 代卖、配置单报价提升成交效率 |
| 数据真，更新快 | 热搜、指数、日报、政策预警、文审动态增强行业权威 |
| 信用强，可成交 | 厂家微官网、营业执照、成交记录、平台收款和有限担保 |
| 圈子活，流量稳 | 行业圈子、达人、区域话题和私聊提高用户留存 |
| 拉新准，转化高 | 潜在会员库持续挖掘、去重、触达、转化和清理 |

---

## 3. 用户身份

平台主身份保持简单，避免权限混乱。

| 主身份 | 说明 |
|---|---|
| 个人用户 | 可查信息、买设备、卖二手、发布商机、参与圈子 |
| 厂家 | 包含整机厂家、配件供应商、贸易商、外地代理、5G 电玩运营商 |
| 厂家业务员 | 由厂家绑定权限，可客服、报价、管理询盘 |
| 服务商 | 周边服务人员或团队，如装机技工、送货师傅、维修师傅 |

达人、业务精英、场地老板、外贸商等不单独作为主身份，可通过认证标签、会员等级和权限表达。

## 4. 产品终端

| 终端 | 定位 |
|---|---|
| 微信小程序 | 买家快速查询、配置单转发、商机浏览、圈子互动、裂变传播 |
| 独立 APP | 深度数据、厂家工作台、业务员跟进、服务商接单、订单收款 |
| Web 管理后台 | 审核、数据治理、产品管理、商机管理、定价开关、潜客转化 |

---

## 5. 阶段目标

### 阶段 1：数据和信任

- 建立厂家库、产品库、文审库、配件库。
- 支持基础搜索、AI 客服、AI 识图。
- 厂家可生成基础微官网。
- 基础信息限时免费，收费模块价格设置为 0。

### 阶段 2：工具和商机

- 上线 AI 一站式选品。
- 支持配置单、厂家一键报价、报价保密。
- 上线商机平台、联系方式低价查看、商家买单主动询价。
- 建立圈子基础版。

### 阶段 3：交易和增长

- 平台收款、有限担保、协议签署。
- 配件商城商家自营。
- 潜在会员库转化系统。
- 厂家推广、达人合作、日报和热搜。

### 阶段 4：行业基础设施

- 官方配件商城。
- 行业指数、深度报告、政策预警。
- 5G 电玩专区、线上探店、营销自动化。

---

## 6. 成功标准

- 用户能快速查到靠谱厂家和产品。
- 买家愿意用配置单进行询价和比价。
- 厂家愿意完善微官网和产品资料。
- 商机有效率和联系方式付费率可持续。
- 圈子内容能带来持续打开和互动。
- 潜在会员能低成本转化为注册用户。
