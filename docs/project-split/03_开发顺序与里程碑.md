# 游艺圈开发顺序与里程碑

版本：V0.1  
日期：2026-07-02

---

## 1. 开发原则

1. 先数据，后交易。
2. 先轻工具，后重闭环。
3. 先免费拉新，后逐步收费。
4. 先人工审核兜底，后 AI 自动化。
5. 小程序负责裂变和快速查询，APP 负责深度经营。

---

## 2. MVP 阶段

目标：验证行业用户是否愿意使用平台查信息、看商机、生成配置单和联系厂家。

必须做：

| 模块 | MVP 功能 |
|---|---|
| 厂家库 | 厂家录入、服务标签、营业执照、认证状态 |
| 产品库 | 产品录入、分类、外观图、游戏画面图、文审状态 |
| 搜索 | 关键词搜索、热搜、自动补全 |
| AI 客服 | 文本和图片输入，找厂家、找产品、查文审 |
| AI 识图 | 设备图、二维码、铭牌识别 |
| 选品工具 | 预算、面积、机台数、退彩票/退礼品筛选 |
| 商机 | 发布、审核、浏览、联系方式隐藏 |
| 厂家微官网 | 基础展示、产品列表、询价入口 |
| 定价系统 | 所有收费模块价格可设置为 0 |
| 后台 | 审核、分类、标签、水印、定价开关 |

可选做：

- 圈子基础版
- 配置单转发报价
- 潜在会员导入和去重
- 配件商城基础
- 每日简报基础版

暂缓：

- 完整担保交易
- 官方配件商城智能分单
- 大规模私聊关系链
- 复杂用户画像
- 大规模自动营销触达

---

## 3. 阶段 1：数据与查询基础

目标：建立可信数据底座。

任务：

- 厂家库
- 产品库
- 文审库
- 配件分类
- 后台审核
- 基础小程序
- 基础搜索
- 热搜词
- 基础埋点
- 定价开关
- 水印配置

验收：

- 可录入厂家和产品。
- 可搜索和筛选。
- 可配置分类和标签。
- 所有收费功能可配置价格为 0。

---

## 4. 阶段 2：AI 工具与商机

目标：形成用户使用理由。

任务：

- AI 客服
- AI 识图
- 商机发布和查看
- 联系方式付费查看
- 商家主动询价服务
- 一站式选品
- 配置单导出
- 厂家一键报价
- 圈子基础版

验收：

- 用户可生成配置单。
- 厂家可收到报价页面并填写报价。
- 商机可审核、隐藏联系方式、查看付费。
- 圈子可发帖、评论、举报。

---

## 5. 阶段 3：厂家和生态

目标：让厂家和服务方愿意长期入驻。

任务：

- 厂家微官网
- 厂家认领
- 厂家业务员权限
- 区域厂家和周边服务地图
- 配件商城商家自营
- 达人主页
- 业务精英橱窗
- 潜在会员库基础版

验收：

- 厂家能管理产品和询盘。
- 业务员能被厂家授权。
- 周边服务商能展示和接单。
- 潜客能导入、去重、标记状态。

---

## 6. 阶段 4：交易与信用

目标：建立平台成交闭环。

任务：

- 平台收款
- 有限担保
- 电子协议签署
- 订单状态
- 提现规则
- 投诉和退款
- 厂家信用评分
- 报价记录沉淀
- 达人推广归因

验收：

- 买家能通过平台支付。
- 厂家达到条件后提现。
- 文审整机可在线签协议。
- 投诉和争议可处理。

---

## 7. 阶段 5：行业权威与增长

目标：形成行业入口和商业化能力。

任务：

- 每日简报
- 行业指数
- 政策预警
- 5G 电玩专区
- 番禺线上探店
- 官方配件商城
- 圈子运营体系
- 潜客转化自动化
- 营销服务和广告系统

验收：

- 日报能自动汇总内容。
- 圈子热点能进入热榜和日报。
- 潜客转化效果可统计。
- 广告和推广可配置价格。
