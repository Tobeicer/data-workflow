# 游艺圈端侧与后台开发任务

版本：V0.1  
日期：2026-07-02

---

## 1. 微信小程序

定位：买家入口、快速查询、配置单转发、圈子互动、裂变传播。

功能：

- 首页搜索
- 热搜词
- 查厂家
- 查产品
- AI 识图
- 查文审
- 商机浏览
- 联系方式查看
- 配置单生成和转发
- 厂家报价页面
- 厂家微官网浏览
- 配件搜索
- 圈子浏览、发帖、评论
- 私聊提醒
- AI 客服

小程序不承担复杂后台管理，只做轻操作和提醒。

---

## 2. 独立 APP

定位：深度经营、厂家工作台、业务员工具、服务商接单、数据工具。

功能：

- 小程序全部核心能力
- 厂家产品管理
- 厂家微官网管理
- 询盘管理
- 报价管理
- 订单和收款
- 提现
- 推广管理
- 业务员权限
- 服务商接单
- 圈子深度互动
- 行业指数
- 数据报告
- 潜客转化辅助

## 5. 关键权限

| 身份 | 权限重点 |
|---|---|
| 个人用户 | 查询、发布商机、卖二手、圈子、配置单 |
| 厂家 | 产品、微官网、询盘、报价、收款、推广 |
| 厂家业务员 | 由厂家配置报价、客服、产品、询盘权限 |
| 服务商 | 服务展示、接单、评价、收款 |
| 平台运营 | 审核、配置、投诉、数据、定价 |

---

## 6. 测试重点

- 搜索准确性
- AI 识图匹配
- 配置单报价隔离
- 联系方式付费
- 价格为 0 流程
- 图片联系方式识别
- 圈子审核
- 私聊限频
- 潜客去重
- 订单支付和提现
