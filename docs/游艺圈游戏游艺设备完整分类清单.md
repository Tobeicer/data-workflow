## 附录 C：包含与排除边界规则

范围判断固定使用三态：`included`、`excluded`、`review_required`。规则结果必须保存规则版本、命中条件、理由和证据；无法确定时进入复核，不生成正式主分类。

### 明确包含

符合以下任一条件即属于采集范围：

1. 标准主分类（A-F）中任一品类。
2. 商品标题、描述或参数中明确标注游戏游艺设备、电玩设备、游乐设备、动漫游戏设备。
3. 主要用途为商业运营场所（电玩城、儿童乐园、商场、景区）的互动娱乐设备。
4. 设备零配件属于 C 零配件或 B 核心总成与套件，且适配机型在适配机型字典（M01-M25）范围内。

### 明确排除

符合以下任一条件即不属于采集范围：

1. 普通家用消费电子产品（Switch、PS5、Xbox、手机、平板、电脑及其配件）。
2. 普通儿童玩具（不具备电控/机械互动机制的塑料玩具、毛绒玩具摆件、积木）。
3. 普通体育器材（非电控的篮球架、乒乓球桌、跑步机等家用健身器材）。
4. 普通家具和家电。
5. 食品、饮料、日用品（D 类耗材中的糖果零食等除外，仅采集明确标注为游艺设备奖品的）。
6. 纯虚拟商品（游戏点卡、充值卡、Steam 游戏 CDK、网游道具）。
7. 租赁服务类商品（如抖音生活服务中的"游戏机租赁"）。
8. 家用迷你版设备明确标注"家用玩具"且不具投币/扫码商用功能的（如掌上迷你抓娃娃机玩具）。

### 边界判定流程

商品同时具备包含和排除特征时，按以下顺序判定：

1. 商品标题、类目、参数中明确属于清单 A-F → 包含。
2. 商品标题明确标注"家用"且无任何商用/运营特征 → 排除。
3. 商品所属平台类目在附录 A 映射表中 → 包含。
4. 其余 → 标记 `review_required`，人工判定后补充规则；不能直接归入 A14。

### 待判定项

> 以下为边界模糊的典型场景，后续开发时逐项决策：

- 普通玩具与游艺设备二合一的商品（如带计分功能的投篮玩具）。
- 既用于家用也用于商用的设备（如家用/商用均可的台球桌）。拼多多/京东大量"家用抓娃娃机"实际具备投币功能，需按是否支持投币/扫码判定。
- VR 一体机是否作为消费电子排除。
- 打印耗材中哪些属于游艺设备专用（拍贴机相纸等）哪些属于通用办公耗材。
- "文审机""中性机"是行业术语，指经文化主管部门审批的商用游戏机，搜索时作为关键词使用，但本身不是独立品类。
