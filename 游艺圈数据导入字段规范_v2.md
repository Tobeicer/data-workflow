# 游艺圈平台消费导入字段规范 v2

更新日期：2026-07-13

状态：当前临时 L3 Excel 兼容格式，待平台源码或正式接口契约确认

本文只规定当前交付文件的字段和格式。SKU、规格、价格库存快照、来源平台、来源商品 ID、采集时间、关系和完整证据仍保存在 L0-L2，不因 v2 未接收而删除。

职责和写库边界以 `docs/游艺圈数据工作流总纲.md` 为准。

## 1. 当前交付范围

- 厂家 XLSX：使用本规范第 2 节的 18 列兼容格式，待平台源码或正式接口契约提供后再完成最终适配。
- 配件 XLSX：沿用本规范的 9 列兼容格式。
- 产品和文审资料尚未形成当前可验证模板，不在本文展开。

L3 文件由平台导入、校验和审核；数据工作流不直接写正式业务表。

## 2. 厂家表

当前表头共 18 列，列名保持不变：

| 列名 | 格式 | 说明 |
|---|---|---|
| 厂家名称 | 文本，必填 | 当前导入显示名称；来源方标签与工商主体未核验时不得伪装成工商全称 |
| 简称 | 文本 | 搜索和显示简称 |
| 地区 | `省/市/区` | 例如 `广东省/广州市/番禺区` |
| 主营品类 | 逗号分隔 | 使用当前白名单 |
| 主营产品 | 文本 | 一句话描述主要产品或服务 |
| 联系人 | 文本 | 公开经营联系人 |
| 联系电话 | 文本 | 公开经营电话 |
| 微信 | 文本 | 公开经营微信；没有则留空 |
| 官网 | 完整 URL | 厂家或店铺公开主页 |
| 地址 | 文本 | 公开经营或注册地址，按来源语义填写 |
| 简介 | 建议 100 字内 | 只写当前已确认主体/品牌信息，不列候选方案和争议过程 |
| 来源链接 | 完整 URL | 用于人工溯源 |
| 成立年份 | 四位整数 | 1900–2100 |
| 厂房面积 | 整数 | 平方米，不带单位和千分位；只映射 `factory_building_area_sqm` |
| 资质认证 | 逗号分隔 | 只填可追溯的来源认证 |
| 服务能力 | 逗号分隔 | 使用当前白名单或经确认词项 |
| 状态 | 枚举 | `已通过`、`待审核`、`已驳回`；不确定时使用 `待审核` |
| 认证等级 | 枚举 | 未完成平台认证时使用 `unverified` |

空值留空，不写 `NULL`、`无`、`-` 或 `0`。

“工厂面积”和“厂房面积”是两个来源语义字段。当前 L3 v2 的“厂房面积”列只映射 `factory_building_area_sqm`，不得使用 `factory_area_sqm` 回填；来源只提供“工厂面积”时，本列留空，`factory_area_sqm` 继续保存在 L1-L2，等待后续正式契约增加独立消费字段。

当前漫立方交付构建器的个别表头和配件分类投影仍可能与本文不一致，因此既有交付只是历史兼容交付，不作为 v2 自动合规证明。实施正式 L3 adapter 时必须通过表头、枚举、空值、键和记录级回执测试。

### 地区格式

| 情况 | 写法 |
|---|---|
| 三级齐全 | `广东省/广州市/番禺区` |
| 只确认到市 | `广东省/广州市/` |
| 直辖市 | `上海市/上海市/浦东新区` |

省、市不可省略；无法确认省市时进入复核，不写“未知”占位。

### 当前主营品类

```text
游戏设备,管理系统,周边配件,机台销售,产品研发,场地规划,
维修服务,二手买卖,合作开店,装修设计,安装调试,其他服务
```

### 资质与服务

资质候选：

```text
CCC认证,CE认证,ISO9001,A级纳税人,高新企业
```

服务候选：

```text
来图加工,来样加工,支持打样,可开专票,支持定制,一件代发
```

来源平台的“超级工厂、金牌制造、店铺等级、回头率、履约率”等只作为带来源和时间的 L2 信号，不直接转成游艺圈认证等级。

## 3. 配件表

| 列名 | 格式 | 说明 |
|---|---|---|
| 厂家名称 | 文本 | 与厂家交付名称一致；无法确认可留空并进入复核 |
| 配件名称 | 文本，必填 | 来源商品名称或清洗名称 |
| 配件分类 | 单选 | 使用当前白名单 |
| 型号 | 文本 | 没有则留空 |
| 最低价 | 数字 | 不带货币符号 |
| 最高价 | 数字 | 不得小于最低价 |
| 配件描述 | 文本 | 可汇总材质、规格、功能和场景 |
| 图片地址 | HTTPS URL | 必须是自有或可长期访问的交付地址 |
| 来源链接 | URL/来源标识 | 用于溯源 |

价格规则：

- 单一价格：最低价和最高价填写相同数字。
- 只有起价：最高价留空。
- 未知价格：两列都留空，不填 0。
- SKU 价格和库存完整保存在 L1/L2；v2 只消费商品级区间。

当前配件分类：

```text
币器/投币器,主板/程序板,摇杆按钮,读卡器,屏幕显示,
电源线材,支付模块,彩票纸/耗材,礼品机配件,礼品
```

## 4. 图片

- 不直接交付易失效的外部电商图床链接。
- 转存后提供可公开访问的 HTTPS 地址。
- 建议 JPG/PNG、长边不超过 1000px、单张不超过 500KB。
- 转存失败时留空并进入质量报告，不保留死链。

## 5. 文件格式

| 项 | 要求 |
|---|---|
| 格式 | `.xlsx` |
| 表头 | 第一行，列名与本文一致 |
| 数据 | 从第二行开始 |
| 空值 | 空单元格 |
| 数字列 | Excel 数字类型，不使用文本数字 |
| 文件拆分 | 厂家和配件分别交付 |

## 6. 契约升级

平台源码、导入 API 或正式模板到达后，核对真实列名、必填、枚举、长度、唯一键、图片流程和错误回执，并发布新版本。升级只修改 L3 适配，不删除 L0-L2 来源字段。
